README file for the HAR run_analysis ScriptJhon J. Sanabria Description of the Script used to tidy the HAR data and summarize itThere is only one scrip included in the run_analysis and all data reading, transformation, selection and pivoting, as well as the file writing, are performed with that single script, that performs the following steps: Set the working directory to the folder with the dataLoad libraries to be used: dplyr and tidyrRead the activity names from the files providedRead the labels for the measurement variables, from the files providedRead the training dataRead the test dataCombine data sets to create the Human Activity Recognition Data FrameRemove temporary objectsSubset Mean and Standard Deviation Data OnlyApply descriptive names for the activities, rearrange and sortCreate files for the Tidy Data Set and for the Summary Data Set
